## JS 精度缺失问题

### 现象

```javascript
0.1 + 0.2 = 0.30000000000000004;

7 * 0.8 = 5.6000000000000005;

var b = 342201195009247816;
b.toString();
```

对于整数，前端出现问题的几率可能比较低，毕竟很少有业务需要需要用到超大整数，只要运算结果不超过 Math.pow(2, 53) 就不会丢失精度。
